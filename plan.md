Paragraph 1 — Locked foundation. Shape-first wiring is in production: each kinetic body owns one collision primitive (CircleShape or convex PolygonShape), and mass, inertia, broadphase, sprite cache keys, and vector overlay are derived from it — no rect side channel. Proof assets (crate, box_2x4, custom_box, tri_wedge, hex_block, beach ball) share initWorldPropShape; draw extrudes the same footprint through the cached polygon bake (flipper/pipe remain documented mesh exceptions with polygon collision). Poly–poly SAT contact helpers and initial separation/normal tests are landed. This layer is the invariant: no new parallel geometry representations, no regressions to box footprints or legacy collision fields.

Paragraph 2 — Convex interaction parity (current arc). Before any geometry mutation, arbitrary convex polygons must behave as reliably as spheres under the existing sequential impulse solver. Remaining work is narrowphase validation and boundary integration: contact tests with asserted separation normals and post-solve penetration clearance, polygon–static-wall resolution without circle-radius shortcuts on elongated footprints, and consistent sleep thresholds plus Coulomb friction on proof-shape stacks and slides. Concave silhouettes and compound rigid bodies stay deferred — decompose into 2–3 convex polygons on one entity first. Fracture is gated on this: stacking, wall contact, and resting behavior on proof props must be boring before subdivision code runs.

Paragraph 3 — Procedural fracture via in-place shape mutation (endgame). Opt-in per asset (fracture: true). A fixed world-space characteristic edge length (POXEL_TARGET_EDGE, invariant to spawn scale) drives outline triangulation and edge subdivision into a parts[] adjacency mesh for draw topology and split connectivity. Simulation stays one convex PolygonShape per body; above an impulse threshold, connected-component extraction over triangle adjacency partitions the mesh — the largest component rewrites the parent shape in place and recomputes mass/inertia; detached pieces spawn through the same initWorldPropShape path as the parent archetype, so fragments and spawned instances share one entity contract. Iteration continues until every edge is below the minimum — a remeshing stop condition, not a separate shard type or rect fallback. Draw uses per-triangle fill from the polygon recipe; fracture mutates only shape and parts. Deprecated splittable/poxel code is reference only for re-porting onto this wiring.
