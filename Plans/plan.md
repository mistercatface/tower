## Snake death rules trilogy (predator–prey + split + inert fracture)

**PR 1 — Snake lifecycle FSM + split-on-impact (game rules).** Introduce a head-scoped `SnakeInstance` registry (`alive` / `inert` / `dead`) owned by `setupSnakeGame`, with min-length death (≤ head + one segment → kill). After the kinetic contact pass, scan head–chain or head–head impacts between distinct snakes; when relative impact speed exceeds a config threshold and the striker is larger (segment count or chain radius), break the victim’s link at the struck segment, keep the head-adjacent subchain alive if it still meets min length, and spawn a new **inert** instance for the severed tail (no autosim, no `chainHead`, links among tail segments optionally kept briefly). Stopping point: deterministic split at link index, autosim stops on death, `snakeSplit.test.js` + `snakeMinLengthDeath.test.js`.

**PR 2 — Predator–prey intent (AI + pathfinding).** Extend `createSeekExploreIntent` (or a snake-specific wrapper) to **`seek_food | seek_prey | flee | explore`**, scoring visible goal orbs vs other snake heads via `queryGridCellVision`. Larger snakes pursue nearest visible smaller head; smaller snakes flee — first slice as flee target = point away from threat + memory stamp on escape corridor (reuse nav step penalty pattern); pursue = HPA seek on prey head. Config: `preySizeRatio`, `huntPriority`, `fleeRange`. Stopping point: multi-snake scenes show hunt/chase/flee without food-only behavior; `snakePredatorPrey.test.js`, `snakeIntent.test.js` extended, `snakePerfBudget` still green.

**PR 3 — Inert corpses + impact fracture (physics gameplay).** Inert segments stay kinetic and chain-linked until secondary hard impacts. Wire contact callbacks (pre-impact speed from existing fracture path) to break remaining tail links and/or spawn ball debris via `propFracture`-style splitting for segment props — game-scoped **break impulse** on distance constraints, not full editor breakable-link trilogy. Predator ram damage uses normal circle contact impulses (no CCD). Stopping point: severed tails crumple and fragment believably; `snakeInertFracture.test.js`; physics.md notes game break threshold as in-scope for snake, editor breakable links remain v2.
